# Norheim
A simple beautiful dual screen front-end for the Ayn Thor. 

## Features
- Works with all Android emulators.
- Media Scraping via SteamGridDB.
- Retro Achievement support.
- Dark / Light theme, as well as a few colours!

## Installation
- Download the APK from the releases tab & install it.
- SteamGridDB account for media scraping.
- IGDB account for meta data scraping.
- Retro Achievements account for achievements where supported.
